Chargement bloqué

Djassa n’a pas pu démarrer. Une extension (AdBlock/Privacy/Antivirus), un cache ou un mauvais type MIME peut bloquer les scripts de l’application.

  1. Essayez un rechargement forcé (Ctrl+Shift+R / Cmd+Shift+R)
  2. Essayez une fenêtre privée
  3. Désactivez temporairement les extensions pour lovable.app

Diagnostic: si vous voyez ce message, le bundle principal n’a probablement pas été exécuté.
